jQuery Spinner Control
Default Implementation|Time Spinner|Spinning List of Items|Decimal Spinner
Plug-in Name: jQuery Spinner Control
Developer: Anant Anand Gupta | MindzGroup Technologies
Description: Developed this plug-in to Spin the given values in text type input.
Options (Parameter Object)
  • type:
    It tells the plugin that what is the type of the control to be rendered. Currently this plugin supports 'range' and 'list' values for this option.
    default value:= 'range'
  • typedata:
    It provides the additional values which will be used to render the specific type of the control. This option has the following subset of options:
    • min:
      minimum value in the range type of the control.
      default value:= 1
    • max:
      maximum value in the range type of the control.
      default value:= 10
    • interval:
      spin interval the range type of the control.
      default value:= 1
    • decimalplaces:
      number of the decimal places in the range type of the control.
      default value:= 0
    • list:
      list of items to spin in the list type of the control.
      default value:= ''
  • defaultVal:
    It tells the plugin that what is the default value shown on the control. default value:= 0 (for the range type of the control)
  • width:
    what is the width of the value display area in px. default value:= '30px'
  • backColor:
    what is the background color of the display area in px. default value:= '#fff'
  • looping:
    if the value reaches to the either of the ends in the provided range/list it starts over from the other end on setting the value to this option as true. default value:= false

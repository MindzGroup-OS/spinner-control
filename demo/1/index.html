Demo: Default Implementation
Demo Description: Simplest implementation of the plug-in. Create an input control of type text (html) or a TextBox (asp.net) on the page. On document ready call the function on jQuery object. If selector returns multiple matches, this function will only be applied to the first one in the array. See the example below.
Code: Default Implementation 
$(function () {
    $("#txtDefault").SpinnerControl();
})

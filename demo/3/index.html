Demo: Spinning List of Items
Demo Description: You can provide a pre-decided comma separated values to spinner. In this demo below we are going to demonstrate the jQuery Spinner Control to spin the Month Names and Week days in two different controls. Also this will give you a chance to see the looping feature of the control.
Code: Spinning List of Items 
$(function () {
    $("#txtMonths").SpinnerControl({ type: 'list',
        typedata: {
            list: 'January,February,March,April,May,June,July,August,September,October,November,December'
        },
        width: '100px',
        looping: true 
    });
    $("#txtMonths").SpinnerControl({ type: 'list',
        typedata: {
            list: 'Monday,Tuesday,Wednesday,Thursday,Friday,Saturday,Sunday'
        },
        width: '100px',
        looping: true 
    });
})
Output:
Months:
Weeks:

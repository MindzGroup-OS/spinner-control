Demo: Decimal Spinner
Demo Description: the control can also be used to spin the values containing the decimals. In this demo below we are going to demonstrate the jQuery Spinner Control to spin the decimal number.
Code: Decimal Spinner 
$(function () {
    $("#txtMonths").SpinnerControl({ type: 'range',
        typedata: {
            min: 0,
            max: 2,
            interval: .2,
            decimalplaces: 2
        },
        width: '100px'
    });
})
